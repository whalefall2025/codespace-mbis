# Finance Reading Assistant

Frontend-only university prototype for helping Chinese-speaking accounting and finance students read English academic passages.

## Run The Demo

```bash
npm install
npm run dev
```

Open the local Vite URL shown in the terminal, usually `http://localhost:5173/`.

## Test And Build

```bash
npm test
npm run build
```

## Demo Scope

- Browser-only React/Vite app.
- Routes: `/` and `/results`.
- Mocked Simple Chinese translation and analysis.
- Temporary `sessionStorage` for the current session result.
- Fixed accounting and finance glossary.
- No backend, database, accounts, saved history, uploads, real AI API, Natural Chinese mode, mobile-optimized layout, or extra routes.
